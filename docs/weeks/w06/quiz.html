12.1 Cara mengerjakan

  • Jawab hanya dengan mengisi fungsi q01() s.d. q12() di submissions/w06/answers.py.
  • Kamu boleh memakai Python untuk menghitung, misalnya lewat submissions/w06/laporan.qmd.
  • Jangan ubah nama fungsi.

12.1.1 Format jawaban

  • T/F: kembalikan True (Benar) atau False (Salah).
  • MC: kembalikan huruf "A", "B", "C", atau "D".
  • Numeric: kembalikan bilangan int atau float (pakai titik . untuk desimal).
